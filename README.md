# 🎓 Student Performance ML Analysis

## 📌 Overview
This project analyzes the impact of demographic and socio-economic factors on student academic performance using machine learning models.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 🤖 Models Implemented
- Logistic Regression
- Random Forest
- K-Nearest Neighbors (KNN)

## 📊 Evaluation
- Accuracy Score
- F1 Score
- Confusion Matrix
- Cross Validation
- GridSearchCV

## 🚀 Future Improvements
- Handle class imbalance with SMOTE
- Convert to regression
- Deploy using Streamlit
